本项目为太极项目的网页展示子项

依赖：
在命令行输入npm install 安装所需依赖

数据库问题：
项目代码中使用了本地的mysql数据库，以备份为user.sql文件
可使用user.sql  恢复数据库

使用方法：
运行：在命令行中键入 node app.js 开启服务
debug：在vscode中选择nodejs调试

主要功能：
用户系统
web三维动画
模型格式转换

技术栈：
nodejs（express、mysql、nodemailer、body-parser）
webgl（three.js）
vue
mysql
